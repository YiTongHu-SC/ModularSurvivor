# Utils - 工具类库

## 概述
提供项目中通用的工具类、扩展方法和辅助功能。

## 主要工具类

### MathUtils
- **职责**：数学计算工具
- **功能**：
  - 2D向量计算
  - 距离和角度工具
  - 插值和缓动函数

### ObjectPool
- **职责**：对象池管理
- **功能**：
  - 游戏对象复用
  - 内存优化
  - 自动回收机制

### EventExtensions
- **职责**：事件系统扩展
- **功能**：
  - 事件快捷方法
  - 安全事件调用
  - 事件链式操作

### DebugUtils
- **职责**：调试工具
- **功能**：
  - 可视化调试
  - 日志格式化
  - 性能监控

## 设计原则
- 静态方法为主
- 无副作用的纯函数
- 良好的性能表现
- 易于单元测试
