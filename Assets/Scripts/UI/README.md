# UI - 用户界面

## 概述
管理游戏的所有用户界面，包括游戏内UI和菜单系统。

## 子模块

### Game/
- **作用**：游戏内UI组件
- **包含**：
  - 波次信息显示
  - 生命值条
  - 游戏状态UI

### Menus/
- **作用**：菜单和设置界面
- **包含**：
  - 主菜单
  - 暂停菜单
  - 设置界面

## 设计特点
- 基于Unity UI Toolkit或UGUI
- 响应式布局设计
- 事件驱动的数据更新
- 支持多分辨率适配

## Demo阶段UI需求
- 显示当前波数
- 显示剩余敌人数量
- 简单的游戏状态指示
- 基本的调试信息显示
