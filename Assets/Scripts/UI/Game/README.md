# Game UI - 游戏内界面

## 概述
负责游戏进行中的所有UI显示，包括游戏状态、波次信息和实时数据。

## 主要组件

### WaveInfoPanel
- **职责**：显示波次相关信息
- **内容**：
  - 当前波数
  - 剩余敌人数量
  - 波次进度

### GameStatusHUD
- **职责**：游戏状态抬头显示
- **内容**：
  - 玩家生命值
  - 游戏时间
  - 击杀统计

### DebugInfoPanel
- **职责**：开发调试信息（可选显示）
- **内容**：
  - FPS显示
  - 系统状态
  - 性能指标

## 实现要点
- 通过事件系统接收数据更新
- 使用数据绑定模式
- 支持实时刷新
- 响应式布局适配
