# Menus - 菜单系统

## 概述
管理游戏的各种菜单界面，包括主菜单、暂停菜单等非游戏内UI。

## 主要组件

### MainMenu
- **职责**：游戏主菜单
- **功能**：
  - 开始游戏
  - 设置选项
  - 退出游戏

### PauseMenu
- **职责**：游戏暂停菜单
- **功能**：
  - 继续游戏
  - 返回主菜单
  - 设置选项

### SettingsMenu
- **职责**：游戏设置界面
- **功能**：
  - 音效音量
  - 画面设置
  - 控制设置

## Demo阶段
在demo-v1中，菜单系统可以非常简化：
- 基本的开始按钮
- 简单的暂停功能
- 重启游戏选项
