# Waves - 敌人波次系统

## 概述
管理敌人的生成、波次控制和难度递增，实现经典的生存游戏体验。

## 子模块

### Systems/
- **作用**：波次系统核心逻辑
- **包含**：
  - WaveSystem（波次主控制器）
  - DifficultySystem（难度系统）
  - WaveController（波次流程控制）

### Data/
- **作用**：波次配置数据
- **包含**：
  - 波次配置
  - 敌人生成规则
  - 难度曲线配置

### Spawners/
- **作用**：敌人生成器
- **包含**：
  - 各种生成模式
  - 生成位置计算
  - 生成时机控制

## 设计特点
- 定时刷怪机制
- 敌人从四周向玩家靠近
- 波次间隔和难度递增
- 灵活的配置系统
