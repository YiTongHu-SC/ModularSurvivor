# Wave Data - 波次配置数据

## 概述
定义波次系统的所有配置数据，支持设计师灵活调整游戏难度和节奏。

## 主要数据类

### WaveConfig
- **职责**：单个波次配置
- **属性**：敌人类型、数量、生成间隔、持续时间

### DifficultyConfig
- **职责**：难度曲线配置
- **属性**：难度增长函数、属性倍数、特殊规则

### EnemySpawnRule
- **职责**：敌人生成规则
- **属性**：生成概率、位置规则、组合模式

### WaveSequence
- **职责**：波次序列配置
- **属性**：波次列表、循环规则、特殊波次

## 配置特点
- 基于ScriptableObject
- 支持随机和固定配置
- 可视化编辑器支持
- 运行时热更新
