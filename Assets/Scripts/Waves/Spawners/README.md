# Spawners - 敌人生成器

## 概述
负责在2D逻辑世界中生成敌人，并管理生成的时机和位置。

## 主要组件

### BaseSpawner
- **职责**：生成器基类
- **功能**：
  - 通用生成逻辑
  - 生成位置计算
  - 生成队列管理

### EdgeSpawner
- **职责**：边缘生成器
- **功能**：
  - 从地图边缘生成敌人
  - 避开玩家视野
  - 均匀分布算法

### TimedSpawner
- **职责**：定时生成器
- **功能**：
  - 按时间间隔生成
  - 生成速率控制
  - 批量生成支持

### FormationSpawner
- **职责**：编队生成器
- **功能**：
  - 特定编队生成
  - 组合单位生成
  - 特殊波次支持

## 生成策略
- 远离玩家的安全距离
- 考虑地形和障碍物
- 支持多种生成模式
