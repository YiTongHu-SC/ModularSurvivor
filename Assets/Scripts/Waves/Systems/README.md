# Wave Systems - 波次系统核心

## 概述
实现波次生成、难度控制和游戏节奏管理的核心逻辑。

## 主要类

### WaveSystem
- **职责**：波次系统主控制器
- **功能**：
  - 波次生命周期管理
  - 敌人生成调度
  - 波次状态跟踪

### DifficultySystem
- **职责**：动态难度调整
- **功能**：
  - 根据波次递增难度
  - 敌人数量和强度scaling
  - 生成间隔调整

### WaveController
- **职责**：波次流程控制
- **功能**：
  - 波次开始/结束条件判断
  - 波次间隔管理
  - 游戏状态同步

## 核心机制
- 敌人从地图边缘生成
- 直线移动向玩家靠近
- 波次清空后开始下一波
- 动态调整敌人属性
