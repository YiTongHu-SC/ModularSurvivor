﻿﻿# Core - 核心系统

## 概述
提供项目的基础功能和通用工具，包括坐标系统、单位管理和事件系统。

## 子模块

### Coordinates/
- **作用**：2D逻辑坐标与3D世界坐标的转换
- **包含**：
  - 坐标转换工具
  - 2D数学运算
  - 区域和碰撞检测工具

### Units/
- **作用**：单位数据结构和基础管理
- **包含**：
  - UnitData（单位基础数据）
  - 单位状态管理
  - 单位工厂类

### Events/
- **作用**：全局事件系统
- **包含**：
  - 事件管理器
  - 游戏事件定义
  - 事件监听器接口

### Input/
- **作用**：统一输入管理系统
- **包含**：
  - **InputManager**：输入管理器单例，支持多平台输入
  - **InputEvents**：完整的输入事件定义
  - **InputSystem_Actions**：Unity InputSystem自动生成的输入动作
  - **混合模式**：事件驱动（按键按下/释放）+ 状态查询（移动方向）
- **支持的输入**：移动、攻击、跳跃、冲刺、交互等游戏动作
- **多平台**：键盘、手柄、触屏的统一API接口

## 设计要点
- 所有核心类都是纯C#类，不依赖Unity MonoBehaviour
- 提供稳定的API给其他模块使用
- 支持单元测试
