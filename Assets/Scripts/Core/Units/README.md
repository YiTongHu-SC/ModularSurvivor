# Units - 单位系统

## 概述
定义游戏中所有单位（玩家、敌人）的基础数据结构和管理功能。

## 主要功能
- **单位数据**：生命值、攻击力、移动速度等属性
- **状态管理**：单位的生存状态、buff/debuff状态
- **单位创建**：单位工厂和配置系统
- **单位更新**：单位数据的统一更新机制

## 核心类设计
- `UnitData`：单位基础数据结构
- `UnitStats`：单位属性和状态
- `UnitFactory`：单位创建工厂
- `UnitManager`：单位集合管理器

## 设计原则
- 数据与逻辑分离
- 支持不同类型单位的统一管理
- 便于序列化和配置化
