# Combat Systems - 战斗系统服务

## 概述

包含所有战斗相关的核心逻辑系统，采用纯C#服务类设计。

## 主要类

### CombatSystem

- **职责**：战斗系统的主控制器
- **功能**：
    - 管理所有活跃的战斗单位
    - 协调攻击和伤害系统
    - 处理战斗状态转换

### AttackSystem

- **职责**：自动攻击逻辑
- **功能**：
    - 自动寻找攻击目标
    - 计算攻击时机和冷却
    - 处理不同攻击类型（射线、弹道）

### DamageSystem

- **职责**：伤害计算和应用
- **功能**：
    - 伤害值计算
    - 护甲和减伤计算
    - 单位死亡处理

### BuffSystem

- **职责**：增益和减益效果管理
- **功能**：
    - 应用和移除Buff/Debuff
    - 处理Buff持续时间和效果叠加
    - 与其他系统交互（如伤害加成）

## 设计原则

- 无状态服务设计
- 依赖注入和接口抽象
- 易于测试和扩展
