# Combat - 战斗系统

## 概述
实现核心战斗机制，包括自动攻击、伤害计算和战斗逻辑处理。

## 子模块

### Systems/
- **作用**：战斗系统服务类
- **包含**：
  - CombatSystem（战斗主系统）
  - AttackSystem（攻击系统）
  - DamageSystem（伤害系统）

### Data/
- **作用**：战斗相关数据结构
- **包含**：
  - 武器数据
  - 技能数据
  - 战斗配置

### Views/
- **作用**：战斗表现层组件
- **包含**：
  - 攻击特效
  - 伤害显示
  - 战斗动画

## 设计特点
- 战斗逻辑完全在2D坐标系中计算
- 自动攻击机制（类似幸存者游戏）
- 支持多种攻击类型（射线、弹道等）
- 可配置的战斗参数
