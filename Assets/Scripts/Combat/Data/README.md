# Combat Data - 战斗数据

## 概述
定义所有战斗相关的数据结构，支持配置化和序列化。

## 主要数据类

### WeaponData
- **职责**：武器属性定义
- **属性**：攻击力、攻击速度、攻击范围、攻击类型

### SkillData
- **职责**：技能属性定义
- **属性**：技能效果、冷却时间、消耗、作用范围

### CombatConfig
- **职责**：战斗系统配置
- **属性**：全局战斗参数、平衡性数值

### AttackPattern
- **职责**：攻击模式定义
- **属性**：攻击轨迹、伤害分布、视觉效果

## 特点
- 支持ScriptableObject配置
- 运行时可动态修改
- 便于数值策划调整
