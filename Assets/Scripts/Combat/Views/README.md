# Combat Views - 战斗表现

## 概述
负责战斗系统的视觉表现，将2D逻辑数据转换为3D视觉效果。

## 主要组件

### UnitView
- **职责**：单位的视觉表现
- **功能**：
  - 将2D逻辑位置映射到3D Transform
  - 播放攻击和受伤动画
  - 显示生命值条

### AttackEffectView
- **职责**：攻击特效显示
- **功能**：
  - 射线攻击的激光效果
  - 弹道攻击的子弹轨迹
  - 爆炸和命中特效

### DamageFloatingText
- **职责**：伤害数字显示
- **功能**：
  - 浮动伤害数字
  - 暴击效果
  - 治疗效果

## 设计原则
- 只负责表现，不包含游戏逻辑
- 通过unitId绑定逻辑对象
- 支持对象池优化性能
