# Assets - 项目资源总览

## 概述
本目录包含ModularSurvivor项目的所有资源文件，采用模块化结构组织，便于管理和协作开发。

## 目录结构

### 代码模块
- **Scripts/**：所有C#脚本代码
  - 采用"2D逻辑，3D表现"的架构设计
  - 模块化系统设计，低耦合高内聚

### 资源模块
- **Prefabs/**：游戏预制体资源
- **Materials/**：材质资源
- **Models/**：3D模型资源  
- **Sounds/**：音频资源
- **Configs/**：配置数据文件

### Unity系统
- **Scenes/**：场景文件
- **Settings/**：项目设置
- **Resources/**：运行时加载资源
- **Plugins/**：第三方插件

## Demo-V1目标
当前处于demo-v1开发阶段，目标是实现：
1. 基础的2D战斗逻辑系统
2. 敌人波次生成系统
3. 简单的3D视觉表现
4. 基础UI显示

## 架构原则
- **逻辑与表现分离**：游戏逻辑在2D坐标系运行，Unity 3D仅用于视觉表现
- **模块化设计**：各系统独立开发，通过接口和事件通信
- **配置驱动**：游戏数值和行为通过ScriptableObject配置
- **可测试性**：核心逻辑类不依赖Unity MonoBehaviour

## 开发指导
1. 新功能优先考虑在Scripts/对应模块中实现
2. 保持资源文件的命名一致性
3. 使用README.md文档记录模块功能
4. 遵循既定的代码规范和架构原则
