# Models - 3D模型资源

## 概述
存放项目中使用的3D模型文件，在demo阶段主要使用Unity基础几何体。

## 模型分类

### 单位模型
- **Units/**：单位相关模型
  - 在demo阶段使用Unity原生Cube/Capsule
  - 玩家：蓝色立方体
  - 敌人：红色立方体（不同大小区分类型）

### 环境模型
- **Environment/**：环境模型
  - 地面：大型平面（Plane）
  - 边界：简单的几何体围栏

### 占位模型
- **Placeholders/**：占位模型
  - 用于快速原型开发
  - 简单几何形状组合

## Demo阶段策略
- 不导入外部模型文件
- 完全使用Unity内置几何体
- 通过材质颜色区分不同对象
- 重点验证逻辑而非视觉效果

## 未来规划
当进入美术制作阶段时：
- 导入专业建模软件制作的模型
- 建立统一的建模标准
- 优化模型性能和LOD
