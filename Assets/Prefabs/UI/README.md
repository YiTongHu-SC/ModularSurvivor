# UI - UI预制体

## 概述
存放所有用户界面相关的预制体。

## 预制体列表

### 游戏内UI
- **GameHUD.prefab**：主游戏界面
  - 波次信息面板
  - 生命值显示
  - 游戏状态指示器

- **WaveInfoPanel.prefab**：波次信息面板
  - 当前波数显示
  - 剩余敌人计数
  - 进度条

### 菜单UI
- **MainMenu.prefab**：主菜单界面
  - 开始游戏按钮
  - 设置按钮
  - 退出按钮

- **PauseMenu.prefab**：暂停菜单
  - 继续游戏
  - 重新开始
  - 返回主菜单

## 设计原则
- 响应式布局
- 统一的视觉风格
- 清晰的层级结构
- 易于本地化

## Demo阶段
- 使用Unity默认UI组件
- 简洁的文字显示
- 基础交互功能
- 重点在数据展示
