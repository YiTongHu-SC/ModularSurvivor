# Environment - 环境预制体

## 概述
存放游戏场景和环境相关的预制体。

## 预制体列表

### 基础场景
- **Ground.prefab**：游戏地面
  - 大型平面
  - 基础材质
  - 碰撞体设置

- **GameBoundaries.prefab**：游戏边界
  - 隐形墙体
  - 限制单位移动范围
  - 敌人生成边界标记

### 环境元素
- **Arena.prefab**：竞技场环境
  - 完整场景组合
  - 包含地面和边界
  - 光照设置

## Demo阶段要求
- 简单的几何体组合
- 纯色材质即可
- 重点在功能而非美术
- 确保性能优化

## 未来扩展
- 添加环境装饰
- 多样化的地形
- 动态环境元素
