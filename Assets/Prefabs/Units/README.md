# Units - 单位预制体

## 概述
存放所有游戏单位的预制体，包括玩家和各类敌人。

## 预制体列表

### 玩家单位
- **Player.prefab**：主角单位
  - 包含UnitView组件
  - 自动攻击逻辑绑定
  - 移动控制（如适用）

### 敌人单位
- **Enemy_Basic.prefab**：基础敌人
  - 简单AI行为
  - 向玩家移动
  - 基础攻击能力

- **Enemy_Fast.prefab**：快速敌人
  - 更高移动速度
  - 较低生命值
  - 适合后期波次

## 组件配置
每个单位预制体应包含：
- UnitView（表现层组件）
- Collider（碰撞检测）
- 适当的视觉模型

## Demo阶段
使用简单的几何体作为占位符：
- 立方体代表不同单位
- 不同颜色区分类型
- 基础的动画效果
