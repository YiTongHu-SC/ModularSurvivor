# Effects - 特效预制体

## 概述
存放所有游戏特效相关的预制体，包括战斗特效、环境特效等。

## 预制体列表

### 战斗特效
- **LaserBeam.prefab**：激光攻击特效
  - 射线攻击的视觉表现
  - 持续时间短
  - 简单的光线渲染

- **BulletTrail.prefab**：弹道轨迹特效
  - 弹道攻击的轨迹
  - 物理抛物线
  - 碰撞检测

- **HitEffect.prefab**：命中特效
  - 攻击命中时的效果
  - 简单粒子系统
  - 快速闪烁效果

### UI特效
- **DamageText.prefab**：伤害数字
  - 浮动伤害显示
  - 渐变消失动画
  - 不同颜色表示伤害类型

## Demo阶段实现
- 使用基础的粒子系统
- 简单的颜色和形状
- 重点在功能完整性
- 避免过于复杂的效果
