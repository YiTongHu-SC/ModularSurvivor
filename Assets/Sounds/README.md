# Sounds - 音频资源

## 概述
存放游戏中使用的所有音频文件，包括音效和背景音乐。

## 音频分类

### 战斗音效
- **Combat/**：战斗相关音效
  - 攻击音效（激光、子弹）
  - 命中音效
  - 单位死亡音效

### 界面音效
- **UI/**：用户界面音效
  - 按钮点击音效
  - 菜单切换音效
  - 警告提示音效

### 环境音效
- **Ambient/**：环境音效
  - 背景音乐
  - 环境氛围音
  - 波次开始/结束音效

### 系统音效
- **System/**：系统提示音效
  - 波次变化提示
  - 游戏状态变化
  - 成就解锁音效

## Demo阶段
- 可以暂时不添加音效
- 或使用Unity Asset Store免费音效
- 重点在功能验证而非音效体验

## 音频标准
- 格式：推荐使用OGG或WAV
- 采样率：44.1kHz
- 比特率：适当压缩以节省空间
- 长度：音效尽量简短
