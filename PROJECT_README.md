# ProjectModularSurvivor - 模块化生存游戏

## 项目概述
一款采用"2D逻辑，3D表现"架构的模块化生存游戏，类似Vampire Survivors的游戏机制，但采用现代化的软件架构设计。

## 技术特色
- **2D逻辑引擎**：所有游戏逻辑在2D坐标系中运行，确保性能和可预测性
- **3D视觉表现**：使用Unity 3D系统提供丰富的视觉效果
- **模块化架构**：系统间松耦合，便于扩展和维护
- **配置驱动**：游戏数值和行为完全可配置化
- **服务导向**：核心逻辑采用纯C#服务类，支持单元测试

## 项目结构
```
Assets/
├── Scripts/           # 所有C#代码
│   ├── Core/         # 核心系统（坐标、单位、事件）
│   ├── Combat/       # 战斗系统
│   ├── Waves/        # 波次系统
│   ├── Backpack/     # 装备系统（预留）
│   ├── UI/           # 用户界面
│   └── Utils/        # 工具类
├── Prefabs/          # 预制体资源
├── Materials/        # 材质资源
├── Models/           # 3D模型
├── Sounds/           # 音频资源
├── Configs/          # 配置文件
└── Scenes/           # Unity场景
```
